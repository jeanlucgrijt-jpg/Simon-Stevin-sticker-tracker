# Simon-Stevin-sticker-tracker
Simon Stevin sticker tracker made by ReAcCie with help from WebCie
-ReAcCie25-26 "Unsinkable"

----------------------------- Important image info -----------------------------

Image data is stored on the congressus server and is pulled on a site via a link. 
It is important to link the image in our database via a ID but there is no need to store the picture there (relevant for both uploading and displaying)

Example: 
        <meta property="og:image" content="https://congressus-simonstev.s3-eu-west-1.amazonaws.com/files/03fd5f07a95549f28e7121e0a65ab614.png"/>
